# BobsBurgersAPI

Para instalar y ejecutar BobsBurgersAPI, sigue estos pasos:

1.Clonar el repositorio
```bash
  npm install
```
2. Instalar dependencias
- Desde una terminal en la carpeta del proyecto, ejecuta:
```bash
  npm install
```
3. Ejecutar el proyecto
```bash
  npm start
```
